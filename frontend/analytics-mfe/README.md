# CityAI — Analytics MFE

This microfrontend provides the municipal staff dashboard, community advocate dashboard, AI chatbot interface, and real-time trend insights.

## Features
- **Staff Dashboard:** High-level metrics, heatmap, and backlog management.
- **Advocate Dashboard:** Community-focused trend analysis and issue tracking.
- **AI Chatbot:** LangGraph-powered assistant for natural language queries.
- **Trend Detection:** AI-driven identification of emerging municipal issues.

## Tech Stack
- React 19 + Vite
- Tailwind CSS
- Apollo Client (connecting to `analytics-service`)
- Leaflet (for heatmaps)
