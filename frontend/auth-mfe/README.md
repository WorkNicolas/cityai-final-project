# CityAI — Auth MFE

This microfrontend handles user registration, login, and profile management for residents, staff, and advocates.

## Features
- **Secure Login:** JWT-based authentication via HTTP-only cookies.
- **Registration:** Multi-role registration flow.
- **OAuth:** Integration for Google and GitHub sign-in.

## Tech Stack
- React 19 + Vite
- Tailwind CSS
- Apollo Client (connecting to `auth-service`)
