## Detailed Rubric

### MongoDB Database
- 0–59: Incorrect model, errors  
- 60–69: Some missing/incorrect fields  
- 70–79: Correct fields, minor issues  
- 80–100: Excellent design and implementation  

### GraphQL API
- 0–59: Incomplete, errors  
- 60–69: Some missing functionality  
- 70–79: Mostly correct  
- 80–100: Excellent design, proper patterns  

### Frontend Design & Implementation
- 0–59: Incomplete, errors  
- 60–69: Poor UI, not aligned with specs  
- 70–79: Mostly correct, UI needs improvement  
- 80–100: Excellent UI and implementation  

### Code Quality (Naming, Structure)
- 0–59: Poor naming, inconsistent  
- 60–69: Some issues  
- 70–79: Mostly correct  
- 80–100: Excellent consistency and clarity  

### AI Usage
- 0–59: Not implemented  
- 60–69: Not working properly  
- 70–79: Mostly correct  
- 80–100: Accurate, efficient, well-integrated
